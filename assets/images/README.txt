Thư mục này dùng để chứa ảnh cho project Friend.

Bạn có thể:
- Kéo/thả ảnh từ máy vào đây.
- Đặt tên file theo ý bạn (ví dụ: avatar.jpg, friends.png, ...).
- Chỉnh lại đường dẫn trong file index.html cho trùng với tên file ảnh.

Ví dụ:
Nếu ảnh tên "avatar.jpg" thì trong HTML dùng:
assets/images/avatar.jpg
